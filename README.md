# kawaz-backend
This repository contains the backend service.

Added a minimal Node.js + TypeScript starter scaffold.

Quick start:

```bash
npm install
npm run dev   # start dev server with ts-node-dev
npm run build # compile to ./dist
npm start     # run compiled code
```
